### 使用方法
* 创建方法
```objc
_foldClock = [[XLFoldClock alloc] init];
_foldClock.frame = self.view.bounds;
[self.view addSubview:_foldClock];
```

* 设置时间
```objc
_foldClock.date = [NSDate date];
```
